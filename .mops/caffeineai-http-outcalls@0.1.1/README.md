# caffeineai-http-outcalls
HTTP outcalls package for [Caffeine AI](https://caffeine.ai?utm_source=caffeine-skill-mops&utm_medium=referral).

HTTP outcalls performed by the backend canister (not in the frontend).

## Install

```
mops add caffeineai-http-outcalls
```
